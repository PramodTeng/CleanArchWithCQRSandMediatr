# Quick Reuse Guide: AutoMapper Configuration

## Quick Copy-Paste Solution

Yes! You can easily reuse this AutoMapper configuration in other applications. Here's what you need to copy:

1. **The Interface** (create in your project):

```csharp
public interface IMapFrom<T>
{
    void Mapping(Profile profile) => profile.CreateMap(typeof(T), destinationType: GetType());
}
```

2. **The MappingProfile Class** (create in your project):

```csharp
public class MappingProfile : Profile
{
    public MappingProfile()
    {
        ApplyMappingsFromAssembly(Assembly.GetExecutingAssembly());
    }

    private void ApplyMappingsFromAssembly(Assembly assembly)
    {
        var mapFromType = typeof(IMapFrom<>);
        var mappingMethodName = nameof(IMapFrom<object>.Mapping);
        bool HasInterface(Type t) => t.IsGenericType && t.GetGenericTypeDefinition() == mapFromType;

        var types = assembly.GetExportedTypes().Where(t => t.GetInterfaces().Any(HasInterface)).ToList();
        var argumentTypes = new Type[] { typeof(Profile) };

        foreach (var type in types)
        {
            var instance = Activator.CreateInstance(type);
            var methodInfo = type.GetMethod(mappingMethodName);

            if (methodInfo != null)
            {
                methodInfo.Invoke(instance, new object[] { this });
            }
            else
            {
                var interfaces = type.GetInterfaces().Where(HasInterface).ToList();
                if (interfaces.Count > 0)
                {
                    foreach (var @interface in interfaces)
                    {
                        var interfaceMethodInfo = @interface.GetMethod(mappingMethodName, argumentTypes);
                        interfaceMethodInfo?.Invoke(instance, new object[] { this });
                    }
                }
            }
        }
    }
}
```

3. **Register in Startup/Program** (add to your DI configuration):

```csharp
services.AddAutoMapper(Assembly.GetExecutingAssembly());
```

## How to Use

1. Create your DTOs/ViewModels implementing `IMapFrom<T>`:

```csharp
public class ProductDto : IMapFrom<Product>
{
    public int Id { get; set; }
    public string Name { get; set; }
}
```

That's it! The mapping will be automatically configured.

## What It Does (Without the Details)

This code:

- Automatically finds all classes that implement `IMapFrom<T>`
- Sets up AutoMapper mappings for them
- No need to manually configure each mapping
- Works with any entity-to-DTO mapping in your application

## Custom Mappings (Optional)

If you need custom mapping, override the Mapping method:

```csharp
public class ProductDetailDto : IMapFrom<Product>
{
    public void Mapping(Profile profile)
    {
        profile.CreateMap<Product, ProductDetailDto>()
            .ForMember(d => d.FullInfo, opt => opt.MapFrom(s => $"{s.Name} - {s.Description}"));
    }
}
```
